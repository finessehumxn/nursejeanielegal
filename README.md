# Nurse Jeanie Legal — Legal Nurse Consulting

Professional website for Jeanie Vatelia, BSN RN — Legal Nurse Consultant based in California.

🌐 **Live site:** [nursejeanielegal.com](https://nursejeanielegal.com)

---

## About

Nurse Jeanie Legal is a California-based Legal Nurse Consulting practice serving plaintiff and defense attorneys remotely. This site showcases three expert roles, transparent pricing, services, Jeanie's story, attorney inquiry form, and a nurse team application form.

## Features

- Full single-page site — sticky nav, mobile hamburger menu, smooth scroll
- Hero with logo (Nurse Jeanie Legal brand mark — genie lamp + scales + stethoscope)
- Three expert roles with rates ($150 / $200 / $350 per hour)
- Transparent pricing section aligned to LNC industry standards
- Eight core LNC services including IME support and life care planning
- Case types section (PI, med mal, WC, wrongful death, elder abuse, toxic torts)
- Personal story section — Jeanie's origin and why she does this work
- Attorney inquiry form (mailto — no backend required)
- Nurse team application form (mailto — no backend required)
- Fully responsive — mobile, tablet, desktop
- Brand palette: deep purple, navy-black, nude/blush, blue

## Tech stack

Pure HTML/CSS/JS — zero runtime dependencies.

- Google Fonts: Cormorant Garamond + DM Sans
- Tabler Icons (CDN)
- `serve` for Railway static hosting
- Logo embedded as base64 PNG

## Deploy to Railway

1. Go to [railway.app](https://railway.app) → New Project → Deploy from GitHub repo
2. Select this repo — Railway auto-detects `railway.json` and deploys
3. Add custom domain in Railway settings

## Deploy to GitHub Pages

1. Repo Settings → Pages → Source: Deploy from branch → `main` / root
2. Site live at `https://finessehumxn.github.io/nursejeanielegal`
